Viewing Tool Accessibility

The tools information page (see figure below) displays information about all the available tools, the user that owns them, the group they belong to, the execution server on which they are installed and the type of tool (conversion or scientific) that they are. You must belong to a group that the tool is assigned to and the execution server (Bourreau) that it is installed on must be available in order to use the tool.

Viewing Tool Statistics

At the bottom of the tools information page there is a link to Tool Statistics. This page displays the tools that you have used and the tools that a particular server has executed by all users in both percentage and quantity.
