Registering Files on CBRAIN


A user has access to MainStore, MindStore and Incoming. Data files can also be uploaded to CBRAIN using SFTP (Secure File Transfer Protocol). Connect using any SFTP client for UNIX, Mac or Windows on host mindstorm.cbrain.mcgill.ca using the port number 7500. Your username and password are those of the CBRAIN website. Upload your files as needed. Then go to the Data Providers tab, find the Data Provider named 'Incoming' and click on it's "Browse" link. Register your files, and they will appear in the file manager. You should MOVE these files to one of the official CBRAIN Data Providers. There is a button at the top of the file manager that will allow you to move files.
