Managing Tasks



The various operations, allowing you to have a certain amount of control over the execution of tasks, are listed below. Queued tasks are those tasks waiting to be executed. Running tasks are those that are currently running on the cluster.
  • Operations for Queued Tasks
    • Hold queued tasks
    • Release held tasks
  • Operations for Running Tasks
    • Suspend running task
    • Resume suspended tasks
  • Operations for All Tasks
    • Terminate selected tasks
    • Remove selected tasks
