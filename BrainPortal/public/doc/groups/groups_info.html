Groups and Projects



Groups facilitate the sharing of files between users. There are four types of groups: system groups, site groups, user groups and work groups.
  • System Groups
    • Every user on CBRAIN is automatically a member of the system group everyone.
  • Site Groups
    • Each site has a site group automatically associated with it.
  • User Groups
    • Every user on CBRAIN is automatically assigned a group, that only they are a member of, specified by that user's username.
  • Work Groups
    • A work group is a type of project that can be created by anyone.
Creating a Project
By selecting the Create Project drop down menu, you can create a new project. The resulting project is a Work Group that only the creator of the group can access. Only a site manager or administrator can assign an additional site and users to these work groups.
